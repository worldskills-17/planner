# Changelog

## [1.0.0] - 2026-04-01

Initial public release.

- Expert roster management with CSV import and inline editing
- Drag-and-drop schedule allocation across teams, floor, and free pool
- Auto-allocation with preference, rotation, and skill balancing
- Marking scheme with modules and sub-criteria
- Overview grid of all experts across all sessions
- Validation engine (hard and soft constraints)
- Auto-save to localStorage, JSON export/import, undo/redo
- WCAG AA accessible, fully offline
